The Quiet Crisis in Cryptography View
RSA and elliptic curve cryptography secure nearly every channel of digital communication, and both are broken by Shor's algorithm on a quantum computer. The collect-now-decrypt-later attack is already underway. The migration to post-quantum standards is not keeping up.

weirwood View
Introducing weirwood: a Rust crate for privacy-preserving XGBoost inference via Fully Homomorphic Encryption. The server evaluates the model on encrypted features and returns an encrypted result — it never sees the input data.
